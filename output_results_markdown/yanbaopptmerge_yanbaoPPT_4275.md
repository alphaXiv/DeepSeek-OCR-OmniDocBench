![A grid diagram showing a red parallelogram and a red rectangle, both occupying the same area.]()![](_page_0_Picture_2.jpeg)

| 平行四边形 | 底  | 高  | 面积               |
|-------|----|----|------------------|
|       | 6m | 4m | 24m <sup>2</sup> |
| 长方形   | 长  | 宽  | 面积               |
|       | 6m | 4m | 24m <sup>2</sup> |

如果长方形的长和宽分别等于平行四边形的底和高，那么它们的面积相等。
