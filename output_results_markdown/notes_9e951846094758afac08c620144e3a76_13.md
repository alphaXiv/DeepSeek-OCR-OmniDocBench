看到“平行”，再看到“周长”  
相似比 = 周长比

∵  $AD \parallel BC$  ∴  $\triangle MAD \sim \triangle MBC$ 

∴ 相似比为  $\frac{AD}{BC} = \frac{8}{10} = \frac{4}{5} = \frac{MA}{MB} = \frac{MD}{MC}$ 

∴  $\frac{\text{Perimeter}(\triangle MAD)}{\text{Perimeter}(\triangle MBC)} = \frac{4}{5}$  ①

∵  $\frac{MA}{MB} = \frac{4}{5}$  ∴  $\frac{MA}{MA+4} = \frac{4}{5}$  ∴  $MA = 16$ 

∵  $\frac{MD}{MC} = \frac{4}{5}$  ∴  $\frac{MD}{MD+5} = \frac{4}{5}$  ∴  $MD = 20$ 

∴  $\text{Perimeter}(\triangle MAD) = MA + AD + MD = 16 + 8 + 20 = 44$  ②

∴ 由①②可知  $\text{Perimeter}(\triangle MBC) = 55$
