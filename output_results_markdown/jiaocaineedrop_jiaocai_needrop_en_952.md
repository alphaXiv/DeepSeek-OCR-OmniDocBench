素呈 $+3$ 价,在进行有关氧化还原反应的计算时,可将 $\text{Fe}$ 元素的化合价看作 $+\frac{8}{3}$ 价。

 $\text{Fe}_3\text{O}_4$ 可以用 $\text{FeO} \cdot \text{Fe}_2\text{O}_3$ 来表示,但 $\text{Fe}_3\text{O}_4$ 是化合物,是一种纯净物,不能将 $\text{Fe}_3\text{O}_4$ 看成是由 $\text{FeO}$ 和 $\text{Fe}_2\text{O}_3$ 组成的混合物。

### 问题3 金属阳离子被还原一定得到金属单质吗?

不一定,存在多种价态的金属可能从较高价态被还原到较低价态,但仍为化合态,如铁粉与氯化铁溶液的反应: $\text{Fe} + 2\text{Fe}^{3+} = 3\text{Fe}^{2+}$ 。

## 应试拓展注意

### 拓展1 铁的氧化物和氢氧化物

#### 铁的氧化物比较

| 名称                                             | 氧化亚铁                                                                          | 氧化铁(俗称铁红)                                                                              | 四氧化三铁(俗称磁性氧化铁)                                                                                    |
|------------------------------------------------|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| 化学式                                            | $\text{FeO}$                                                                  | $\text{Fe}_2\text{O}_3$                                                                | $\text{Fe}_3\text{O}_4$                                                                           |
| 颜色、状态                                          | 黑色粉末                                                                          | 红棕色粉末                                                                                  | 黑色晶体                                                                                              |
| 铁的价态                                           | $+2$ 价                                                                        | $+3$ 价                                                                                 | $+2$ 、 $+3$ 价                                                                                     |
| 水溶性                                            | 均不溶于水                                                                         |                                                                                        |                                                                                                   |
| 与非氧化性酸反应                                       | $\text{FeO} + 2\text{H}^+ = \text{Fe}^{2+} + \text{H}_2\text{O}$              | $\text{Fe}_2\text{O}_3 + 6\text{H}^+ = 2\text{Fe}^{3+} + 3\text{H}_2\text{O}$          | $\text{Fe}_3\text{O}_4 + 8\text{H}^+ = \text{Fe}^{2+} + 2\text{Fe}^{3+} + 4\text{H}_2\text{O}$    |
| 与 $\text{H}_2$ 、 $\text{CO}$ 、 $\text{Al}$ 等反应 | $\text{FeO} + \text{H}_2 \xrightarrow{\Delta} \text{Fe} + \text{H}_2\text{O}$ | $\text{Fe}_2\text{O}_3 + 3\text{CO} \xrightarrow{\text{高温}} 2\text{Fe} + 3\text{CO}_2$ | $3\text{Fe}_3\text{O}_4 + 8\text{Al} \xrightarrow{\text{高温}} 9\text{Fe} + 4\text{Al}_2\text{O}_3$ |

【说明】(1)  $\text{FeO}$ 不稳定,易被氧化为 $\text{Fe}_3\text{O}_4$ 。

(2)  $\text{FeO}$ 、 $\text{Fe}_3\text{O}_4$ 遇氧化性酸(如 $\text{HNO}_3$ )发生氧化还原反应, $+2$ 价的铁均被氧化为 $+3$ 价。

(3)  $\text{FeO}$ 、 $\text{Fe}_2\text{O}_3$ 均为碱性氧化物, $\text{Fe}_3\text{O}_4$ 是复杂的氧化物,不属于碱性氧化物。

| 名称    | 氢氧化亚铁                                                                                                               | 氢氧化铁                                                                                                  |
|-------|---------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| 化学式   | $\text{Fe}(\text{OH})_2$                                                                                            | $\text{Fe}(\text{OH})_3$                                                                              |
