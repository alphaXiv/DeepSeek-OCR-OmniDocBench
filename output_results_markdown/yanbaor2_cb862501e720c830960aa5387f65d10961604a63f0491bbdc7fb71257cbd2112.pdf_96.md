![Diagram illustrating INCHTEK's intelligent vehicle security architecture.]()Diagram illustrating INCHTEK's intelligent vehicle security architecture. Key components include:

- INCHTEK.AI (云驰汽车安全大脑) receives input from TARA工具等咨询技术服务 (TARA tools and consulting technical services) and outputs to 政策法规解读和分析 (Policy interpretation and analysis).
- 云驰智云VSOC平台 (INCHTEK Cloud VSOC Platform) receives input from 政策法规解读和分析 and 工具化传输至VSOC (Tool-based transmission to VSOC).
- The platform interacts with various vehicle domains (Drivetrain, Connectivity, Body&Comfort, ADAS-Autonomy, Infotainment, Gateway, Domain controller) via INCHTEK.SDK and INCHTEK.GW (for OTA升级).
- INCHTEK.APP and INCHTEK.SDK are involved in report transmission.
- The architecture supports various endpoints: 路侧RSU (Roadside RSU), 边缘计算单元 (Edge computing unit), 移动设备 (Mobile device), 自动驾驶车 (Autonomous driving vehicle), 商用车 (Commercial vehicle), and 乘用车 (Passenger vehicle), utilizing INCHTEK.SDK, INCHTEK.APP, and INCHTEK.GW.

## 公司描述

云驰未来秉承着“让人与机器安全和谐相处”的理念，专注于为未来的智能汽车提供全生命周期的信息安全解决方案和产品。具备ECU/DCU信息安全开发、整车信息安全正向开发能力，并结合自动驾驶和软件定义汽车等关键技术，基于软硬结合、车云一体思路，构建智能汽车纵深防御的信息安全体系。作为智能汽车信息安全技术的引领者，云驰未来为众多OEM、自动驾驶公司以及智能网联示范区提供信息安全整体解决方案。已累计为20余家自动驾驶公司、10余家OEM及Tier1提供产品与服务。
